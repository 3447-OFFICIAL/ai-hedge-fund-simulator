# Contributing to AI Hedge Fund Simulator

Thank you for your interest in contributing to the AI Hedge Fund Simulator! We are building a world-class, institutional-grade open-source quantitative platform, and we welcome contributions from engineers, quants, and researchers.

## How Can I Contribute?

### 1. Reporting Bugs
- Use the GitHub Issue Tracker.
- Describe the bug clearly, including steps to reproduce, expected behavior, and actual behavior.
- Include environment details (OS, Python version, dependency versions).

### 2. Suggesting Enhancements
- Open a feature request issue.
- Explain *why* this enhancement is valuable for institutional or quant use cases.

### 3. Submitting Pull Requests
- Fork the repository and create your branch from `main`.
- Write clean, type-hinted Python code.
- Ensure all tests pass using `pytest`.
- Follow the PEP-8 style guide.
- Update documentation and README.md if necessary.
- Write a clear, concise PR description.

## Development Environment Setup

1. Clone your fork: `git clone https://github.com/3447-OFFICIAL/ai-hedge-fund-simulator.git`
2. Create a virtual environment: `python -m venv venv`
3. Install development dependencies: `pip install -r requirements.txt`
4. Set up pre-commit hooks (if configured).

## Code of Conduct
Please be professional, respectful, and collaborative. We maintain the highest standards of open-source engineering.
